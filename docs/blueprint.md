# **App Name**: Golden Heart Hub

## Core Features:

- Daily Prayer Counter: Tracks the completion of daily prayers (Subh/Fajr, Sunrise, Dhuhr, Asr, Maghrib, Isha).
- Dhikr Tracker: Allows users to track their daily Dhikr (remembrance) goal, with customizable increments and a progress bar.
- 99 Names of Allah Mastery: Presents the 99 names of Allah with their meanings.
- Dhikr Mastery: Allows users to manually input Dhikrs for study.
- Day Streak Counter: Tracks the user's consecutive days of completing at least 5 prayers.

## Style Guidelines:

- Primary color: Deep gold (#D4AF37) to represent spiritual value and warmth.
- Background color: Dark navy blue (#1A237E) to provide a contemplative backdrop.
- Accent color: Soft amber (#FFB300) for interactive elements and highlights.
- Body font: 'Literata', a serif font providing legibility, plus a literary and slightly vintage feel.
- Headline font: 'Literata', used to maintain readability throughout.
- Use simple, elegant icons for prayer times and Dhikr milestones.
- Dashboard layout with distinct sections for each practice, mirroring the reference image.
